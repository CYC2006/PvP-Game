# Skills Design Document

冷卻時間記錄於 `chars.csv` 的 `cd_rmb / cd_space / cd_e / cd_r` 欄（秒，`0` = 技能未實作）。  
技能的詳細行為、物理參數、視覺效果記錄於本文件。

## Agent（hitman1）

### Space — 衝刺 Dash　`cd = 5s`
**行為**  
朝當前 WASD 移動方向衝刺約 100px；位移期間忽略 WASD 輸入、禁止 running。  
無移動方向輸入時不觸發。

**物理**  
- 初速 v₀ = 15 px/tick（900 px/s）  
- 每 tick 減速 0.9 px/tick²  
- 速度低於 7.5 px/tick（v₀/2）時停止，共約 9 ticks ≈ 102 px  
- `speed_mult` 透過 UDP 封包送至 server，由 server 執行位移（權威）

**視覺**  
衝刺期間每幀在玩家後方噴出 3 顆灰塵粒子：  
- 顏色：灰米色系 `(190,180,165)` / `(168,160,148)` / `(210,202,188)`  
- 速度：25–70 px/s，方向為衝刺反方向 ± 37°  
- 壽命：0.12–0.28 秒

### E — 閃光彈 Flash Grenade　`cd = 8s`
**行為**  
向滑鼠瞄準方向投擲閃光彈；投擲物移動、停止、短暫延遲後爆炸。  
爆炸範圍內的對手螢幕白屏，不影響投擲者。

**投擲物物理**  
- 初速 v₀ = 8.8 px/tick，方向 = aim 方向  
- 減速 0.2 px/tick²，完全停止（v → 0），射程約 198 px，飛行約 44 ticks（0.73s）  
- 停止後等待 12 ticks（0.2 秒）再爆炸  
- 無視所有實體障礙物（穿透），可停止在障礙物上  
- 不對玩家造成接觸傷害，可穿透玩家

**爆炸**  
- 爆炸半徑 120 px  
- 爆炸位置 = 投擲物最終停止座標  
- 投擲者見到白色爆炸擴散特效（純視覺，不白屏）  
- 對手若在半徑內，觸發白屏效果

**白屏效果（對手）**  
- 總時長 3 秒（180 ticks）  
- 前 1 秒（ticks 180→121）：白色覆蓋層 alpha = 255，看不到任何遊戲畫面  
- 後 2 秒（ticks 120→1）：白色 alpha 從 255 線性降至 0（視力恢復）  
- 白屏期間 HUD（左上資訊、彈藥、技能圓圈、血條）仍可見

**爆炸視覺（投擲者本地）**  
- 擴散白色圓圈：半徑 0 → 130 px，持續 0.5 秒  
- Alpha 230 → 0 隨半徑增大線性衰減

**投擲物視覺**  
- PNG sprite：`assets/objects/flashbang_P1.png` / `flashbang_P2.png`（48×36 px）  
- 旋轉速度與投擲物當前速度成正比（等減速旋轉），停止後角度鎖定

### E — 手榴彈 Grenade　`cd = 8s`
**行為**  
向滑鼠瞄準方向投擲手榴彈；停止後短暫延遲爆炸，對範圍內敵方造成距離衰減傷害。

**投擲物物理**  
- 一次投出 3 顆，每顆間隔 4 ticks（共 ~0.13 秒）  
- 初速 v₀ 各自隨機 8.5–9.5 px/tick（一位小數），減速 0.2 px/tick²，射程約 180–225 px  
- 投擲方向帶 ±10° 隨機偏角  
- 停止後等待 12 ticks（0.2 秒）再爆炸  
- 無視所有實體障礙物（穿透），可停止在障礙物上  
- 不對玩家造成接觸傷害

**爆炸傷害**  
- 爆炸半徑 120 px  
- 傷害依距離二次方衰減：  
  - 中心（dist = 0）：50 damage  
  - 邊緣（dist = 120）：10 damage  
  - 公式：`damage = 10 + 40 × (1 − dist/120)²`
- 投擲者本身不受傷

**爆炸視覺**  
- 擴散彩色圓圈（玩家所屬顏色）：半徑 0 → 130 px，持續 0.5 秒  
- Alpha 200 → 0 隨半徑增大線性衰減  
- P1 = 藍色 `(160,220,255)`，P2 = 橘紅色 `(255,180,140)`

**投擲物視覺**  
- PNG sprite：`assets/objects/grenade_P1.png` / `grenade_P2.png`（48×48 px）  
- 旋轉速度與投擲物當前速度成正比，停止後角度鎖定

## Assassin（survivor1）

### Space — 速度提升 Speed Boost　`cd = 4s`
**行為**  
按下後移動速度提升為 1.5 倍，持續 3 秒（180 ticks）。  
射擊僵直（shoot_slow）在速度提升期間仍然套用（乘算），例如：基礎 × 0.2 × 1.5 = 0.3。

**參數**  
- 持續時間：3 秒  
- 移速倍率：× 1.5  
- 射擊僵直疊加：shoot_slow × boost_mult

**視覺**  
- 持續時間內每 6 tick 在玩家位置留下半透明殘影  
- 殘影顯示 24 tick 後消失（alpha 從 170 線性降至 0）

### E — 未設計　`cd = 0`
> **草稿**：煙霧彈（投擲物，落點產生持續 4 秒的霧區遮蔽視野）

### R — 未設計　`cd = 0`
> **草稿**：替身（放假身，玩家隱形 2 秒）

### RMB — 成長手裡劍 Growing Shuriken　`cd = 5s`
**行為**  
朝滑鼠瞄準方向射出大型手裡劍；飛行中持續等速變大，碰到敵人或地圖邊界消失。  
不受障礙物阻擋（穿透）。

**物理**  
- 速度：800 px/s（≈ 13.3 px/tick），等速  
- 射程：無限（僅地圖邊界為止）  
- 0 spread

**傷害**  
- 碰撞半徑：`r = BULLET_RADIUS + age × 0.3`（px/tick）  
- 傷害（線性）：`damage = 15 + 1.5 × (r − BULLET_RADIUS)`  
  - 出生（age = 0）：15 damage，r = 5 px  
  - 1 秒後（age = 60）：42 damage，r = 23 px  
  - 3 秒後（age = 180）：96 damage，r = 59 px

**視覺**  
- 旋轉 8 點星形（玩家所屬顏色）  
- 半徑隨 age 線性成長，外有發光圓暈
